# Week 07 - Python Data Exploration and Cleaning with Pandas

📌 Objective

The objective of this assignment is to perform basic data exploration and data cleaning using the Pandas library in Python. The assignment also includes creating a derived column, performing basic filtering operations, and generating interactive visualizations.

 📂 Dataset

- **Dataset Name:** Sample - Superstore.csv
- **Tool Used:** Jupyter Notebook
- **Language:** Python
- **Libraries:** Pandas, Plotly

 ✅ Tasks Performed

- Imported the Pandas library.
- Loaded the Superstore dataset.
- Explored the dataset using:
  - head()
  - tail()
  - shape
  - columns
  - dtypes
  - info()
- Checked and handled missing values.
- Performed basic filtering and column selection.
- Removed duplicate records.
- Created a derived column (`Price` and `total_amount`).
- Saved the cleaned dataset as a new CSV file.
- Created interactive graphs using Plotly.
- Added business insights based on the analysis.

 📊 Visualizations

- Sales by Category
- Sales Distribution (Pie Chart)
- Sales vs Profit
- Quantity Sold by Category

 💡 Key Insights

- Technology generated the highest sales.
- Office Supplies recorded the highest quantity sold.
- Some orders had high sales but low profit.
- Data cleaning improved the quality of the dataset.
- Interactive charts helped visualize business performance.
